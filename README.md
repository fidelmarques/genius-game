# Jogo GENIUS

Jogo da memória criado na década de 80 em que, aleatoriamente, será apresentada uma sequência de luzes e o usuário deverá repetir a sequência sorteada.

## Tecnologias:
* HTML5 <br>
* CSS3 <br>
* Javascript Vanilla
